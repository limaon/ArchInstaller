About This Project

This is a collection of shell scripts that automate the installation and configuration of a fully-functional Arch Linux desktop system.

The scripts include:

  • Desktop environment installation (KDE, GNOME, i3, etc.)
  • Network, Bluetooth, audio, and printing support
  • Automated package installation and system configuration

Important Note

This web interface is designed to help you explore and configure the ArchInstaller scripts. The actual installation scripts must be run on a physical machine or VM booted from an Arch Linux ISO.

To use the installer:

  1. Download the Arch Linux ISO from archlinux.org
  2. Boot from the ISO
  3. Clone the repository: git clone https://github.com/limaon/ArchInstaller.git
  4. Run: ./archinstall.sh
